# Usage

The algorithm can be run from the Processing toolbox:

1. Open **Processing ► Toolbox**.
2. Search for **Append Features With Field Mapping**.
3. Double‑click to open the dialog.
4. Set the **Source layer**, **Target layer**, and define the **Field mapping** (e.g., `source_field=target_field`).
5. Click **Run**.

The selected features from the source layer will be appended to the target layer with attributes remapped according to the mapping you provided.
